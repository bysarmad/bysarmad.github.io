Web-based Graphic User Interface Development

This project was performed at University of Stavanger, Norway



The goal of this project is to develop a web application that would serve as a Graphical User Interface for real-time drilling data management and visualization. It would facilitate the use of real-time databases, such as the Volve dataset’s drilling related features, lab data generated by the UiS Drillbotics Team’s lab rig or other third-party data source. The integration of his app into other software, such as the OpenLab drilling simulator would represent a possible way of commercial deployment.
